# 📱 Mini Katalog Uygulaması

Bu uygulama, bir API üzerinden dinamik veri çekerek ürünleri listeleyen, detaylı inceleme sunan ve sepet yönetimi sağlayan modern bir Flutter mobil ticaret kataloğudur.

### 📝 Proje Açıklaması
- **Dinamik Veri:** `wantapi.com` üzerinden REST API entegrasyonu.

### 🛠 Teknik Detaylar
- **Flutter Sürümü:** 3.22.0

### 🚀 Çalıştırma Adımları
1. Bu depoyu klonlayın: `git clone https://github.com/Mertkctp/Katalog_app.git`
2. Proje dizinine gidin: `Katalog_app`
3. Bağımlılıkları yükleyin: `flutter pub get`
4. Uygulamayı çalıştırın: `flutter run`
